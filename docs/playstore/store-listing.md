```
🎭 관극 — 뮤지컬·연극을 사랑하는 모든 분들을 위한 커뮤니티입니다.

■ 주요 기능

[공연 정보]
• 국내 상연 중인 뮤지컬·연극 공연 목록을 한눈에
• 공연별 캐스팅 캘린더로 내가 원하는 배우의 날짜 확인
• 관심 공연 북마크로 업데이트 소식 놓치지 않기

[커뮤니티]
• 공연별 게시판에서 후기, 정보, 질문, 잡담 나누기
• 배우별 게시판에서 팬들과 소통
• 공연 꿀팁, 좌석 정보, 팬 정보 공유

[관람 기록]
• 내가 본 공연, 날짜, 좌석, 캐스팅, 별점, 한줄 메모 기록
• 연간 관람 통계로 나의 관극 히스토리 한눈에

[배우 팔로우]
• 좋아하는 배우를 팔로우하고 관련 글 모아보기
• 홈 피드에서 관심 공연·배우 최신 소식 확인

[안전한 커뮤니티]
• 불쾌한 게시글 신고 및 특정 사용자 차단 기능
• 건전한 공연 문화 커뮤니티 유지

■ 이런 분께 추천드려요
✓ 공연 후기를 나누고 싶은 뮤지컬·연극 팬
✓ 좋아하는 배우의 캐스팅 날짜를 놓치고 싶지 않은 분
✓ 나의 관람 기록을 체계적으로 관리하고 싶은 분
✓ 공연 정보와 꿀팁을 얻고 싶은 분

지금 바로 관극 커뮤니티에 합류하세요! 🎭
```
