계정 삭제 요청

계정을 삭제하면 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다.
앱 내에서도 [내 정보 → 계정 설정 → 회원 탈퇴]를 통해 즉시 탈퇴할 수 있습니다.

⚠️ 삭제되는 데이터를 미리 확인해 주세요. 제출 후 취소가 불가능합니다.

삭제되는 데이터

  • 계정 및 프로필 정보 (닉네임, 프로필 사진)
  • 작성한 게시글 및 댓글
  • 관람 기록 (날짜, 좌석, 별점, 메모)
  • 북마크, 팔로우 정보
  • 좋아요, 신고 기록
위 내용을 모두 확인했으며, 모든 데이터가 영구 삭제됨에 동의합니다.
또는
이메일로 직접 요청: privacy@gwanggeuk.app
