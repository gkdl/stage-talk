회원 탈퇴 및 데이터 삭제 안내

스테이지톡(StageTalk) · 시행일 2026-06-25

스테이지톡 이용자는 아래 방법으로 계정과 모든 데이터를 삭제할 수 있습니다.

1. 앱에서 직접 탈퇴하기

  1. 스테이지톡 앱 실행
  2. 하단 프로필 탭 이동
  3. 설정 > 계정 설정 선택
  4. 회원 탈퇴 선택 후 확인

탈퇴를 확정하면 계정과 데이터가 즉시 영구 삭제되며, 복구할 수 없습니다.

2. 삭제되는 데이터

회원 탈퇴 시 아래 데이터가 모두 즉시 삭제됩니다.

  • 계정 정보(카카오 로그인 연동 식별자)
  • 프로필(닉네임, 프로필 이미지)
  • 작성한 게시글·댓글
  • 관람 기록 및 연결된 배우 정보
  • 좋아요, 북마크, 배우 팔로우
  • 신고·차단 내역

3. 보관되는 데이터

원칙적으로 보관되는 데이터는 없습니다. 단, 관계 법령에 따라 보관 의무가 있는 정보가 있는 경우 해당 법정 기간 동안만 보관 후 파기합니다.

4. 앱 없이 탈퇴를 요청하려면

앱에 접근할 수 없는 경우, 아래 이메일로 가입에 사용한 계정 정보와 함께 탈퇴를 요청해 주세요. 본인 확인 후 처리해 드립니다.
